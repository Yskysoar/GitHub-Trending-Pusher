## 3. 接口实现详情

### 3.1 DashboardService（3个接口）

| 方法 | 返回类型 | 实现说明 |
| --- | --- | --- |
| `get_today_stats()` | `dict` | 从最近一条summary_logs记录获取统计，无记录返回全零值 |
| `get_top_repos(limit)` | `list[dict]` | 查询repositories表eval_score>0的TOP N |
| `get_latest_summary()` | `dict | None` | 查询summary_logs最新一条 |

### 3.2 RuleService（5个接口）

| 方法 | 返回类型 | 实现说明 |
| --- | --- | --- |
| `get_rules(enabled_only)` | `list[dict]` | 获取规则列表，格式化JSON字段为列表，布尔化enabled |
| `add_rule(rule_data)` | `dict` | 校验后调用crud.add_rule，返回格式化后的规则 |
| `update_rule(rule_id, rule_data)` | `dict` | 仅更新非None字段，返回格式化后的规则 |
| `delete_rule(rule_id)` | `None` | 删除规则，不存在时抛出RuleError |
| `toggle_rule(rule_id, enabled)` | `None` | 启用/禁用规则 |

**数据校验**：
- 规则名称不能为空
- 关键词列表不能为空
- 优先级必须在1-10之间

**数据格式化**：`_format_rule()` 将keywords/topics从JSON字符串转为列表，enabled从int转为bool。

### 3.3 HistoryService（6个接口）

| 方法 | 返回类型 | 实现说明 |
| --- | --- | --- |
| `get_summaries(page, size)` | `dict` | 分页获取，返回PaginatedResult结构 |
| `search_summaries(keyword, page, size)` | `dict` | 按关键词搜索title和content |
| `get_summary_detail(log_id)` | `dict | None` | 获取单条日志详情 |
| `delete_summary(log_id)` | `None` | 删除数据库记录+磁盘文件+关联summary_repos |
| `open_file(file_path)` | `None` | 委托FileManager.open_file |
| `open_directory(dir_path)` | `None` | 委托FileManager.open_directory |

**分页结果结构**：
```python
{
    "items": list[dict],
    "total": int,
    "page": int,
    "size": int,
    "total_pages": int,
}
```

### 3.4 SettingsService（8个接口）

| 方法 | 返回类型 | 实现说明 |
| --- | --- | --- |
| `get_settings()` | `dict` | 获取完整配置，敏感信息掩码处理 |
| `save_settings(settings_data)` | `None` | 深度合并配置，保留未修改的敏感字段 |
| `restore_default_settings()` | `None` | 恢复所有配置为默认值 |
| `test_github_connection(token=None)` | `dict` | 调用GitHub /user接口测试，支持传入输入框Token |
| `test_llm_connection(api_key=None)` | `dict` | 调用LLM /chat/completions接口测试，支持传入输入框API Key |
| `set_autostart(enabled)` | `None` | 注册表操作+配置同步保存 |
| `get_evaluation_config()` | `dict` | 获取评估配置段 |
| `save_evaluation_config(config)` | `None` | 保存评估配置，含权重之和校验 |

**连接测试返回结构**：
```python
{
    "success": bool,
    "message": str,
    "latency_ms": int,
}
```

**敏感信息处理**：
- `get_settings()` 返回时对token和api_key进行掩码（`mask_sensitive()`），用于非编辑场景
- `get_settings_for_edit()` 返回原始值，用于编辑表单（输入框已有 `show="•"` 遮盖显示）
- `save_settings()` 时对token和api_key执行 `sanitize_ascii()` 清洗

### 3.5 TaskService（4个接口+3个辅助方法）

| 方法 | 返回类型 | 实现说明 |
| --- | --- | --- |
| `run_task_now()` | `None` | 委托Scheduler.run_task()异步执行 |
| `get_task_status()` | `dict` | 返回运行状态+上次/下次执行时间 |
| `cancel_task()` | `None` | 委托Scheduler.cancel_task() |
| `toggle_autostart(enabled)` | `None` | 委托SettingsService.set_autostart() |
| `set_callback(callback)` | `None` | 设置任务回调 |
| `start_scheduler()` | `None` | 启动定时调度 |
| `stop_scheduler()` | `None` | 停止定时调度 |

**任务状态返回结构**：
```python
{
    "is_running": bool,
    "current_step": str,
    "progress": float,
    "last_run_time": str,
    "next_run_time": str,
}
```
