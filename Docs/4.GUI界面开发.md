### 3.1 主窗口（app.py）

**类**：`App(ctk.CTk)`

**窗口配置**：
- 默认尺寸：1080×720
- 最小尺寸：900×600
- 主题：跟随系统（可切换浅色/深色）

**布局结构**（侧边栏 + 顶部栏 + 主体 + 状态栏）：
```
┌──────────────┬──────────────────────────────────┐
│   侧边栏     │  顶部栏 (52px, 固定, CARD_BG)    │
│  (200px)     │  页面标题          [操作按钮]     │
│  背景:#0d0e12│─────────────────────────────────│
│              │                                  │
│ GH Trending │  主体内容 (可滚动)                │
│ ◈  推送概览  │  (CTkScrollableFrame)            │
│ ◆  推送规则  │                                  │
│ ○  历史记录  │                                  │
│ ⚙  系统设置  │                                  │
│ ─────────── │                                  │
│ ▶  立即执行  │                                  │
│ ⏰ 09:00     │                                  │
├──────────────┼──────────────────────────────────┤
│  状态栏 (28px, 固定, SIDEBAR_BG)                │
└──────────────┴──────────────────────────────────┘
```

**核心功能**：
- 侧边栏导航切换（4个页面，激活状态用ACTION_BLUE高亮）
- 固定顶部栏显示当前页面标题（由_switch_page动态更新）
- 立即执行按钮（调用`scheduler.run_task()`，执行中禁用+文字变化）
- 状态栏实时显示任务进度
- 首次运行引导弹窗（检测Token/API Key为空时自动弹出）
- 关闭窗口最小化到托盘（可配置）

### 3.2 公共UI组件（widgets.py）

**按钮组件**：

| 组件 | 用途 | 样式 |
| --- | --- | --- |
| `PrimaryButton` | 主要操作（保存、执行、确认） | 蓝色实心，白色文字 |
| `SecondaryButton` | 次要操作（取消、浏览、恢复默认） | 透明+边框，深色文字 |
| `DangerButton` | 危险操作（删除） | 红色实心，白色文字 |
| `GhostButton` | 幽灵操作（编辑、切换显示） | 透明无框，悬停变色 |

**布局组件**：

| 组件 | 用途 | 关键属性 |
| --- | --- | --- |
| `SectionCard` | 配置区块卡片 | title + icon + accent_color，3px色条，`body`属性 |
| `FieldRow` | 表单字段行 | label(FONT_LABEL) + widget_frame，统一标签宽度120px |
| `PageHeader` | 页面标题栏 | title + subtitle + right_frame |
| `PageDivider` | 页面分割线 | 1px BORDER色 |

**功能组件**：

| 组件 | 用途 | 关键属性 |
| --- | --- | --- |
| `TimePicker` | 时间选择器 | 小时(00-23) + 分钟(00-55,步长5)下拉框，`get()`/`set()` |
| `LanguageMultiSelect` | 语言多选下拉框 | 点击展开Toplevel+CheckBox列表，支持26种GitHub常用语言，`get()`返回list，`set()`接受list或逗号分隔字符串 |
| `SaveIndicator` | 自动保存指示器 | `show_saved()`/`show_error()`，2秒后自动消失 |
| `TagLabel` | 标签/徽章 | text + color，12%透明度背景+彩色文字 |
| `MessageBox` | 消息弹窗 | icon(check/cancel/info/warning) + message + PrimaryButton确认 |

**数据展示组件**：

| 组件 | 用途 | 关键属性 |
| --- | --- | --- |
| `StatCard` | 统计卡片 | title + value + 3px顶部色条，`set_value()` |
| `RepoListItem` | 仓库列表项 | rank + full_name + TagLabel(stars/score/language) |
| `RuleCard` | 规则卡片 | 关键词TagLabel + GhostButton编辑 + DangerButton删除 |

### 3.4 规则管理页面（rules_page.py）

**类**：`RulesPage(ctk.CTkScrollableFrame)`

**功能**：
- 规则列表展示（RuleCard组件，关键词显示为TagLabel）
- 新建规则弹窗（名称*、关键词*、主题、语言多选、最低Star、优先级滑块+数字、启用开关）
- 编辑规则弹窗（预填现有数据）
- 删除确认弹窗（SecondaryButton取消 + DangerButton删除）

**规则编辑弹窗字段**：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| 规则名称 | Entry | 是 | — |
| 关键词 | Entry | 是 | 逗号分隔 |
| GitHub主题 | Entry | 否 | 逗号分隔 |
| 编程语言 | LanguageMultiSelect | 否 | 多选下拉框，26种GitHub常用语言 |
| 最低Star数 | Entry | 否 | 0=使用全局配置 |
| 优先级 | Slider + Label | 否 | 1-10，左侧数字实时跟随滑块，蓝色强调 |
| 启用规则 | CheckBox | 否 | 默认启用 |

**优先级滑块修复**：
- 数字标签在滑块左侧，使用FONT_BODY_EMPH + ACTION_BLUE
- 滑块command回调实时更新数字：`lambda v: label.configure(text=str(int(v)))`

### 3.6 设置页面（settings_page.py）

**类**：`SettingsPage(ctk.CTkScrollableFrame)`

**5个配置区域**：

| 区域 | 色条 | 字段 | 特殊功能 |
| --- | --- | --- | --- |
| GitHub配置 | ACCENT_TEAL | Token(掩码)+测试、抓取间隔、执行时间(TimePicker)、启用定时任务、最大抓取数、最低Star | PrimaryButton测试、TimePicker |
| 大模型配置 | ACTION_BLUE | 代理厂家选择、API Key(掩码)+测试+获取模型、Base URL、模型选择(下拉+自定义) | PrimaryButton测试/获取模型、Provider切换自动更新Base URL和模型 |
| 推送与评估 | ACCENT_PURPLE | 推送数量、4个权重输入 | 权重之和实时校验 |
| 输出设置 | ACCENT_ORANGE | 日志保存目录 | SecondaryButton浏览 |
| 应用设置 | BRAND_DARK | 主题下拉、最小化到托盘 | 主题切换即时生效 |

**自动保存机制**：
- 所有表单字段输入完成后即刻自动保存（无需点击保存按钮）
- Entry字段：FocusOut/Return事件触发，600ms防抖
- 下拉框/复选框：选择即刻保存
- TimePicker：选择即刻保存
- 保存成功显示"✓ 已自动保存"指示器（2秒后消失）
- 保存失败显示错误信息（3秒后消失）
- 刷新页面时设置`_loading`标志防止触发自动保存

**Provider下拉框**：
- 初始化时从配置中读取所有providers的name列表
- 选择provider时自动更新Base URL和模型下拉框
- 保存时正确映射provider name到provider key

**获取模型**：
- 使用OpenAI兼容的`/models`接口获取用户API Key可用的模型列表
- 提示文案明确说明"加载您可用的模型列表"
- 获取成功后显示"共N个可用模型"

**时间选择器**：
- 替代原来的文本输入框
- 小时下拉(00-23) + 分钟下拉(00-55, 步长5)
- 选择即刻保存

**操作按钮**：SecondaryButton恢复默认
