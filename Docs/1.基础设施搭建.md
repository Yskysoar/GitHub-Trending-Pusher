### 3.3 数据库设计

5张表：

| 表名 | 用途 | 关键约束 |
| --- | --- | --- |
| rules | 推送规则 | name UNIQUE |
| repositories | 仓库数据 | full_name UNIQUE |
| match_records | 匹配记录 | (rule_id, repo_id) UNIQUE, CASCADE删除 |
| summary_logs | 总结日志 | — |
| summary_repos | 日志-仓库关联 | (summary_id, repo_id) UNIQUE, CASCADE删除 |

13个索引：

| 索引名 | 类型 | 表 |
| --- | --- | --- |
| idx_rules_name | UNIQUE | rules |
| idx_rules_enabled | 普通 | rules |
| idx_repositories_full_name | UNIQUE | repositories |
| idx_repositories_fetched_at | 普通 | repositories |
| idx_repositories_stars_growth | 普通 | repositories |
| idx_repositories_eval_score | 普通 | repositories |
| idx_match_records_rule_id | 普通 | match_records |
| idx_match_records_repo_id | 普通 | match_records |
| idx_match_records_rule_repo | UNIQUE | match_records |
| idx_summary_logs_generated_at | 普通 | summary_logs |
| idx_summary_repos_summary_id | 普通 | summary_repos |
| idx_summary_repos_repo_id | 普通 | summary_repos |
| idx_summary_repos_summary_repo | UNIQUE | summary_repos |

### 3.4 配置管理

7个配置段：

| 配置段 | 键 | 默认值 |
| --- | --- | --- |
| github | token, fetch_interval_hours, max_repos_per_fetch, trending_languages, min_stars, growth_period | "", 24, 50, ["python","javascript",...], 100, "daily" |
| llm | provider, base_url, api_key, model, temperature, max_tokens | "volcengine", "https://ark.cn-beijing.volces.com/api/v3", "", "GLM-4.7", 0.7, 4096 |
| evaluation | top_n, weights | 10, {rule_match:0.3, star_threshold:0.2, growth_speed:0.2, learning_value:0.3} |
| output | save_dir, format, filename_template | "", "markdown", "github_trending_{date}.md" |
| scheduler | enabled, run_time, timezone | true, "09:00", "Asia/Shanghai" |
| autostart | enabled | false |
| app | theme, language, minimize_to_tray | "system", "zh_CN", true |

环境变量覆盖：`GITHUB_TOKEN`、`VOLCENGINE_API_KEY`、`APP_DATA_DIR`

**关键设计**：`get_all()` 返回深拷贝（`copy.deepcopy`），防止调用方修改脱敏字段时污染原始配置数据。
